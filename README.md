# S&P 500 GARCH Volatility Dashboard

A time series dashboard for modeling and forecasting S&P 500 volatility using GARCH models.

## Features
- Real-time S&P 500 data fetching via Alpha Vantage API
- GARCH(1,1) volatility modeling
- Interactive volatility forecasting
- Risk metrics (VaR, volatility estimates)
- Docker containerized deployment

## Data Source
- **Index**: S&P 500 (via SPY ETF)
- **API**: Alpha Vantage
- **Frequency**: Daily data

## Tech Stack
- Python 3.12
- ARCH library for GARCH models
- Plotly Dash for interactive dashboard
- Alpha Vantage API for market data
- Docker for containerization
